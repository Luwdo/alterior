instantiationError

  • instantiationError(injector: ReflectiveInjector, originalException: any, originalStack: any, key: ReflectiveKey): InjectionError
    • Defined in packages/di/src/injection/reflective_errors.ts:137

    Thrown when a constructing type returns with an Error.

    The InstantiationError class contains the original error plus the dependency graph which caused this object to be instantiated.

    Example (live demo)

    class A {
  constructor() {
    throw new Error('message');
  }
}

try {
  injector.get(A);
} catch (e) {
  expect(e instanceof InstantiationError).toBe(true);
  expect(e.originalException.message).toEqual("message");
  expect(e.originalStack).toBeDefined();
}
