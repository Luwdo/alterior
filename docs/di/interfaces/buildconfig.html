publishBackend

publishBackend: boolean
  • Defined in src/build-config.ts:14

When publishing your project to NPM, should the backend code be included? If false, only the frontend code (the client portion) will be published to NPM. If true, both the frontend and the backend code will be published, allowing your backend to be run from the package itself.
